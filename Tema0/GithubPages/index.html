Crear página web estática con Github Pages

Crear repositorio

Creamos un nuevo repositorio y le ponemos de nombre "nombredeusuario.github.io" donde nombredeusuario es tu nombre de usuario

Cambiamos configuración del repositorio

Para ello vamos a ir a la pestaña de Configuración de nuestro repositorio recién creado.

Cuando estemos en Configuración nos vamos a Páginas y en el apartado de Compilacion e implementación, en Origen seleccionamos Implementar desde una rama.

Después en Rama podremos elegir la rama de origen de publicación.

Ya estaría creada nuestra página web con Github Pages

Podemos acceder a ella en el enlace https://nombredeusuario.github.io/
